The editor

Formatting that gets out of your way.

Tap to add the next beat. Swipe between scenes. The editor handles the format so you stay in the scene.

  • Element-first editor — tap to append action, dialogue, or a new slug
  • Smart "Then" suggestions guide you from action to dialogue to the next cut
  • Character autocomplete pulls from your project's bible
  • Monospaced script font with automatic uppercase for headings and characters
The bible

Every character, every project, one tap away.

Grouped by project, searchable across everything. Your cast stays with the script it belongs to.

  • Role tags (Protagonist, Supporting, Antagonist, Lead)
  • Bios and arcs travel with the project
  • Add, edit, or delete from the list — no drill-in required
  • Names auto-suggested in dialogue so the flow stays unbroken
