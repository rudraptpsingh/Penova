The Mac workflow

Plan in cards. Outline in lanes.
Write the page.

Four panes that respect the way screenwriters actually work — beats first, dialogue last. Switch between them with one keystroke; everything stays in sync.

Pane 01 — Index cards

Beat board for the structural pass.

Every scene is a card. Color-stripe by beat type — setup, inciting, midpoint, climax. Drag to reorder; the script renumbers everything underneath. Click a card to jump straight into its dialogue.

Pane 02 — Outline

Sortable lanes for the navigational pass.

See every scene's location, time, beat, and page count in one table. Click any column header to re-sort. Hover a row to preview, click to open. Faster than scrolling through a 110-page PDF.

Pane 03 — Editor

Distraction-free pages, with the chrome you need.

Cream-paper page on a dark canvas. Tab cycles element type. Right-click a line for Insert above / Insert below / Change kind / Delete. ⌘⌫ to delete a row regardless of content. ⌘F to search the script. Auto-complete pulls from prior cues and locations.

Pane 04 — Title page & export

Industry-format PDF, FDX, Fountain.

One sheet for title, author, and contact block. Export to WGA-standard PDF, Final Draft (.fdx), or Fountain plain-text — with the title page wired through correctly. Lock the script for production and scene numbers freeze in place (just like Final Draft).
