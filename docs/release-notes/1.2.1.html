Penova 1.2.1

2 May 2026 · macOS 14+

Mac script panel polish

  • Element-kind chip removed entirely. The "ACTION" / "TRANSITION" badge that appeared on the focused row was overlapping right-aligned transition text like "CUT TO:" — and on Mac, the chip was redundant anyway. The script panel now reads cleanly as a real screenplay: transitions right-aligned ALL CAPS, dialogue indented, scene headings full-width. You can still change a row's element kind via ⌘1–⌘7 or the right-click context menu.
  • Window opens wide enough for the paper. The minimum window width has been raised from 1024 → 1280pt so the 640pt screenplay paper always sits centered between the library sidebar and the inspector — no more horizontal scrolling on existing installs whose macOS-saved window frame was at the previous 1024 floor. Default size for fresh installs bumped to 1380×880 for breathing room.
