How updates work

Penova for Mac ships with Sparkle, the standard macOS auto-update framework. The app checks for new versions automatically once an hour while running; when one is available, you'll see a "A new version is available" dialog. You can also force a check via Penova → Check for Updates….

Every released DMG is signed with an Apple Developer ID certificate, notarized by Apple, and verified by an EdDSA signature against the public key bundled in your installed app. No server-side trust required — the entire update path is cryptographic.

Penova for iPhone ships through TestFlight beta currently; App Store launch coming soon.
