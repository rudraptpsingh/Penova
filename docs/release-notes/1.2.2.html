Penova 1.2.2

4 May 2026 · macOS 14+

Patch release that lands the bigger 1.2.0 feature train on top of the live 1.2.1 baseline, plus two bugs that surfaced when the v1.2 PRs were integration-tested end to end.

What's actually new in this patch

  • ⌘K Command Palette reliably opens in every layout. The hidden ⌘K Button was sometimes routed out of the window's responder chain by another .background() modifier; consolidated to a single hidden-shortcuts block.
  • Esc dismiss works on the palette regardless of where focus sits. A window-level cancelAction button catches the keystroke even before the TextField wins focus.
  • End-to-end XCUITest walks every v1.2 surface in one deterministic run — Sprint chip, Command Palette, Save Revision sheet, Beat Board overlay toggle, Search, Title Page, Export. Green on every commit going forward.
  • Mac project management gap closed if you missed the 1.2.0 line: right-click any project in the sidebar for Rename… / Archive / Move to Trash. Trashed and archived projects hide by default; the "Show archived & trash" toggle at the sidebar footer reveals them.
  • Save Revision sheet (⌥⌘R) replaces the silent save with an explicit confirm step + optional note. Walks the full WGA colour rotation — White → Blue → Pink → Yellow → … → Double Cherry, then wraps.
  • Style Check in the inspector flags adverbs, clichés, and passive voice in action lines and parentheticals. Quiet, never blocking.
  • Voiced Table Read (⌥⌘P) reads the script aloud with per-character voices via Apple's on-device speech engine. Premium voices auto-detected when installed.
  • Beat Board structure overlays in Index Cards view — toggle Penova-default / Field 3-Act / Save the Cat / Hero's Journey over the same scenes.
  • Sprint timer in the toolbar — click to start a focused-writing session with live elapsed + word delta.

For the full feature breakdown with rationale, see the 1.2.0 release notes or the launch blog post.

Numbers

  • ~150 new unit tests across the v1.2 PR train
  • Zero regressions
