Penova 1.1.1

2 May 2026 · macOS 14+

Writer-essentials

  • (CONT'D) auto-insertion — when the same character speaks again in a scene without a different character cue in between, the second cue automatically gets (CONT'D) appended. Matches Final Draft's "automatic character continueds" default. Works through suffixes too: JANE (V.O.) → action → JANE becomes JANE (CONT'D).
  • Global character rename — editing a character's name in the Characters tab now rewrites every cue + dialogue/parenthetical reference across every scene of every project the character belongs to. Element-aware: never touches action lines, dialogue body text, or characters whose names happen to share a prefix (so renaming JANE → JEAN leaves JANET alone).
  • Spell check + autocorrect — autocorrect is now properly off on every ALL-CAPS row (heading, character, transition, act break) and on for action / dialogue / parenthetical. No more "INT." getting corrected to "Int." or character names getting "fixed" to common words.

Updates

  • Faster auto-update notifications — Penova checks for new versions every hour while running (was every 24 hours). When a new version is available, the standard "A new version is available" dialog appears automatically — no need to click Penova → Check for Updates… first. The on-demand command is still there if you want to force a check.

Validation

  • Validated end-to-end against three real-world reference screenplays: Big Fish (122 pages, 192 scenes — John August), Brick & Steel and The Last Birthday Card (Stu Maschwitz). Penova's Fountain parser, PDF parser, and FDX reader all handle them cleanly.
  • Validated (MORE) / (CONT'D) markers at page breaks render correctly when dialogue spans pages.
  • 422 unit tests across 43 suites pass (24 new in this release).
